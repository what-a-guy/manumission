Manumission

A command-line utility for building Windows-style HTML help files.

The goal of this project is to supply a free software alternative to the Microsoft Help Workshop. Not only should a developer be able to create a help file outside of Windows, but should also be able to automate the build using the command line.

Why use the CHM format? Because it offers several advantages over other alternatives:

  • Many users are already familiar with this format.
  • All help data is contained in a singe file.
  • Very good CHM viewers already exist on non-Windows platforms.
  • Contents, search and index tabs make the interface easy to use.

Most of the features available to HTML help are planned for manumission, with the exception of the "merge" feature.
